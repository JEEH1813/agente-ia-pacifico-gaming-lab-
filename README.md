# 🖥️ Pacífico Gaming Lab - Agente de IA Corporativo

## 📌 Descripción del Proyecto
Este repositorio contiene el desarrollo de un Agente de Inteligencia Artificial corporativo diseñado para actuar como una base de conocimiento centralizada y conversacional. El agente está optimizado para responder de manera precisa a las consultas de los colaboradores y clientes de **Pacífico Gaming Lab**, una tienda especializada en hardware de alto rendimiento, ensambles personalizados y periféricos gamer.

El proyecto está construido bajo una arquitectura **RAG (Generación Aumentada por Recuperación)** para procesar y comprender múltiples documentos internos en formato PDF.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python 3.x
* **Interfaz de Usuario:** Streamlit (en desarrollo)
* **Framework de IA:** LangChain / LlamaIndex (en desarrollo)
* **Infraestructura de Nube:** Oracle Cloud Infrastructure (OCI)
